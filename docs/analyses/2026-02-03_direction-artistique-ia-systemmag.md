## 🎯 LE PROBLÈME

### Pourquoi l'IA génère des zips traditionnels ?

```
VOTRE DEMANDE              CE QUE L'IA "ENTEND"          CE QU'ELLE GÉNÈRE
─────────────────          ────────────────────          ─────────────────
"zip magnétique            "zip" = 99.9% de son         → Zip avec dents
sur un pull"               training data = zip           → Tirette métal
                           traditionnel YKK              → Fermeture éclair
```

**Le problème statistique :**
- Les modèles IA sont entraînés sur des **milliards d'images de zips traditionnels**
- **0 image** de zip SYSTEMMAG dans le training data (produit breveté unique)
- Quand l'IA lit "zip", elle génère ce qu'elle connaît : dents + tirette

## 🔑 STRATÉGIE DE CONTOURNEMENT

### Principe fondamental

> **NE JAMAIS dire "zip" ou "fermeture éclair" dans les prompts**

### Vocabulaire à utiliser

| ❌ NE PAS DIRE | ✅ DIRE À LA PLACE |
|----------------|-------------------|
| zip magnétique | magnetic closure strip |
| fermeture éclair | magnetic rail system |
| système de fermeture | magnetic snap points |
| zip | concealed magnetic seam |
| tirette | self-aligning magnetic base |
| zip invisible | hidden magnetic junction |

### Négative prompts obligatoires

Toujours inclure dans CHAQUE génération :

```
--no zipper teeth, metal slider, pull tab, interlocking teeth, 
coil zipper, traditional zipper, YKK, zip pull, zipper coils
```

### PHASE 2 : Training LoRA personnalisé (Semaines 2-4)
**Budget : 200-500€ | Temps : 2-3 semaines**

```

├── 20-30 photos du zip SYSTEMMAG
├── Variations : ouvert, fermé, différents tissus, angles
├── Résolution minimum : 1024x1024
└── Qualité : pas de flou, bon éclairage

ÉTAPE 2 : Captioning automatique
├── Utiliser Florence-2 ou GPT-4 Vision
├── Token unique : "SYSMAG_closure" ou "magnzip_v1"
├── Description précise de chaque image
└── Exemple : "SYSMAG_closure magnetic strip on navy fabric, closed position, 5cm spacing"

ÉTAPE 3 : Training sur Replicate ou Leonardo.ai
├── Modèle base : Flux.1 [dev]

├── Rank : 32 (pour produit technique)
├── Epochs : 10-20
└── Learning rate : 1e-4

ÉTAPE 4 : Validation
├── Tester en contextes variés
├── Vérifier pas d'overfitting
└── Documenter les meilleurs seeds
```

## ✅ CHECKLIST QUALITÉ VISUELLE

### Avant publication, vérifier :

#### Structure du produit
- [ ] Aucune dent de zip visible
- [ ] Points magnétiques espacés régulièrement
- [ ] Pas de tirette/slider métallique
- [ ] Bande lisse, pas ondulée ou dentée
- [ ] Forme cohérente avec les vrais produits SYSTEMMAG

#### Qualité d'image
- [ ] Résolution suffisante (min 1080p pour social, 4K pour print)
- [ ] Pas d'artefacts IA visibles (mains déformées, texte illisible)
- [ ] Éclairage cohérent avec la scène
- [ ] Couleurs fidèles à la charte SYSTEMMAG

#### Cohérence marque
- [ ] Vocabulaire respecté dans tout texte associé
- [ ] Pas de référence à une armée spécifique
- [ ] Qualité premium (pas low-cost aesthetic)
- [ ] Message cohérent avec le positionnement (système, pas composant)
