# PHILOSOPHIE DE TRAVAIL

## Principe #1 : Qualité Absolue

> "Si ce n'est pas le meilleur contenu qu'on puisse produire, on recommence."

- Chaque post LinkedIn sera travaillé comme une campagne
- Chaque visuel sera au niveau des plus grands studios
- Chaque texte sera anti-slop (data-driven, preuves, pas de bullshit)

## Principe #2 : Reverse Engineering

Pour chaque type de contenu, on identifie :
1. Les 5 meilleurs exemples qui existent
2. Ce qui les rend excellents
3. Comment faire pareil (ou mieux) pour Eric/SYSTEMMAG

## Principe #3 : IA au Maximum

| Tâche | Outil IA | Résultat |
|-------|----------|----------|
| Visuels produits | Midjourney, Magnific | Qualité studio photo |
| Vidéos courtes | Runway, CapCut | Contenu dynamique |
| Rédaction | Claude, GPT | Itération rapide |
| Voix off | ElevenLabs | Si besoin narration |

## Principe #4 : Itération Rapide

```
CRÉER → PUBLIER → MESURER → AMÉLIORER → RÉPÉTER
```

On ne cherche pas la perfection du premier coup. On publie, on apprend, on améliore.
